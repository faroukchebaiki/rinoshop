# Rinoshop

Rinoshop is a modern marketplace for digital products built on Next.js and
TypeScript. The UI stays intact while the backend is being rebuilt on Neon,
Prisma, and Neon Auth.

## Stack
- Next.js 16 App Router
- React 19 + TypeScript
- tRPC
- Tailwind CSS + shadcn/ui
- Stripe
- Neon Postgres + Prisma (in progress)
- Neon Auth (email/password, in progress)

## Development
```bash
pnpm install
pnpm dev
```
## Environment
Copy `.env.example` to `.env` and fill in values.
